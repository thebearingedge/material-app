material-app
----

#### Development
```bash
λ npm test           # runs all tests
λ npm start          # starts the app
λ npm run lint       # lints all source code
λ npm run cover      # covers unit and integration tests
λ npm run build      # builds the app
λ npm run watch      # builds the app and starts browser-sync at :3001
λ npm run tdd:client # runs all client tests with a watcher
λ npm run tdd:server # runs all server tests with a watcher
λ npm run db:up      # runs new db migrations
λ npm run db:down    # rolls back all db migrations
```
